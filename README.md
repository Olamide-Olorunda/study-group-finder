Study Group Finder

A web-based application that helps students create, join, and manage study groups easily.

To run this project locally, you need:

* XAMPP (or any Apache + PHP server)
* Web browser (Chrome, Firefox, Edge)
* Git (optional, for cloning)

Setup

1. Clone the repository

Or download the ZIP and extract it.

Move project to XAMPP htdocs

Place the project folder inside:

C:\xampp\htdocs\

Start XAMPP

Open XAMPP Control Panel
Start Apache
Start MySQL 

Open phpMyAdmin: http://localhost/phpmyadmin
Create database study_group_finder
Import the provided .sql file
Update database credentials in the PHP config file

Run the project
Open your browser and go to:
http://localhost/study_group_finder/

Make sure Apache is running before accessing the site.
